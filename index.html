Documentation of example_gni_project¶

example_gni_project is a pure Python library for solving initial value problems given by first-order ordinary differential equations (i.e. a vector fields on the state space) and an initial condition (i.e. a point of the state space).

The library was used as part of the Geometric Numerical Integration course offered by the Institute of Applied Dynamics in the summer semester of 2021.
